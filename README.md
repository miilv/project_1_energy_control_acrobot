# Advanced Control Projects

Four self-contained control-design studies. Projects 1-3 share an
underactuated two-link acrobot plant; Project 4 moves to a planar
spacecraft rendezvous problem on Clohessy-Wiltshire dynamics. Each subfolder
holds its own code, plots, animation, README, and notes.

| # | Folder | Plant | Method |
|---|--------|-------|--------|
| 1 | [`project_1_energy_control_acrobot/`](project_1_energy_control_acrobot/) | Acrobot | Energy-shaping swing-up + LQR |
| 2 | [`project_2_adaptive_control_acrobot/`](project_2_adaptive_control_acrobot/) | Acrobot | Adaptive friction control |
| 3 | [`project_3_backstepping_acrobot/`](project_3_backstepping_acrobot/) | Acrobot | Backstepping under actuator dynamics |
| 4 | [`project_4_mpc_docking_spacecraft/`](project_4_mpc_docking_spacecraft/) | Spacecraft (CW) | Constrained MPC for rendezvous + docking |

To run a piece, `cd` into its folder and follow that folder's README.
